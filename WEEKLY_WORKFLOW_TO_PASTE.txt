# Pirelli Weekly Audit MVP V4

Usa **solo questa versione** per aggiornare il repo.

## Cosa aspettarti nel nuovo Excel
Se la versione corretta è davvero in esecuzione, nel workbook troverai:
- foglio `Build Info`
- build/versione `V4_20260331`
- nel foglio `Pages` le colonne `crawl_depth` e `discovered_from`
- nel foglio `Findings` le colonne:
  - `title`
  - `description`
  - `impact`
  - `suggested_fix`
  - `fingerprint`

## Cosa caricare su GitHub
Sovrascrivi questi elementi:
- `main.py`
- `requirements.txt`
- `README.md`
- cartella `audit_engine`
- cartella `config`

## Workflow
Il workflow reale è già incluso nel repo: `.github/workflows/weekly_audit.yml`.
Se vuoi aggiornare/incollare manualmente, usa `.github/workflows/WEEKLY_WORKFLOW_TO_PASTE.txt`.
